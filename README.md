# mars
mars(玛尔斯)
# 简介
mars是基于springboot的微信公众号快速开发框架。使用springboot的自动装配实现『打开即用』，帮助项目过滤掉与微信通信的过程，自动兼容明文/兼容/加密模式，使微信公众号开发更多的关注产品本身业务逻辑。
